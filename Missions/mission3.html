Mission 3 - Active Directory

Accueil Missions Projets Stages

Contexte

Dans le cadre de ma formation BTS SIO, j'ai réalisé la mise en place d'un domaine Active Directory afin de gérer les utilisateurs, les ordinateurs et les droits d'accès dans un réseau d'entreprise.

Objectifs

  • Installer Windows Server
  • Installer Active Directory
  • Créer un domaine
  • Créer des utilisateurs et groupes
  • Mettre en place des GPO
  • Configurer un serveur de fichiers

Matériel et logiciels utilisés

  • Windows Server
  • Active Directory
  • GPO
  • Serveur de fichiers
  • Machine virtuelle

Réalisation

J'ai installé Windows Server puis ajouté le rôle Active Directory. J'ai ensuite créé le domaine et intégré les postes clients au domaine. Des utilisateurs et groupes ont été créés afin de gérer les droits d'accès aux ressources. Des stratégies de groupe (GPO) ont été mises en place pour sécuriser les postes utilisateurs.

Compétences BTS SIO mobilisées

  • Gérer le patrimoine informatique
  • Mettre à disposition un service informatique
  • Administrer un réseau
  • Gérer les habilitations

Résultat

Le domaine Active Directory est fonctionnel. Les utilisateurs peuvent se connecter au domaine et accéder aux ressources selon leurs droits.
